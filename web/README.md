# Ledger Website

Simple static website for the Ledger app.

## Deploy to Vercel

1. Install Vercel CLI (if not already installed):
```bash
npm install -g vercel
```

2. Deploy from the root directory:
```bash
vercel
```

3. Follow the prompts to link your project

The website will be deployed from the `/web` directory.

## Local Development

Open `web/index.html` in your browser to preview locally.

## Content

- **Hero Section**: Main landing area with download link
- **Features**: Key app features
- **Privacy Policy**: Privacy information (update email and details as needed)
- **Contact**: Support email

## Customization

- Update the support email in `index.html`
- Add app screenshots to the hero section
- Update privacy policy details as needed
