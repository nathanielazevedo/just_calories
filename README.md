# Ledger

A simple calorie tracking app built with React Native and Expo.

## Features

- Track daily calorie intake and exercise
- Set weight loss/gain goals
- View weekly projections
- Monitor daily goals with checkboxes
- Weekly weight tracking

## Tech Stack

- React Native
- Expo
- TypeScript
- React Native Paper
- AsyncStorage
